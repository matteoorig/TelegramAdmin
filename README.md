# TelegramAdmin
Telegram bot back-end to learn about events in your area.

###Setup
####Step by step
- creare bot telegram 
    andare su botFather e seguire la registrazione
    salvare il TOKEN
- inserire nella directory principale il file **conf.csv**
    all'interno del file inserire il token
    'token= TOKEN'

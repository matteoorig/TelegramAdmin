# TelegramAdmin
Telegram bot back-end to learn about events in your area.

### Setup
#### Step by step
- creare bot telegram 
    1. andare su botFather e seguire la registrazione
    2. salvare il TOKEN
- inserire nella directory principale il file **conf.csv**
    1. all'interno del file inserire il token
    2. `token= TOKEN`
